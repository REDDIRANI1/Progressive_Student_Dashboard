Authentication

Unified Login Gateway

Single entry point for both students and mentors. The backend validates credentials against bcrypt hashes and returns a signed JWT that encodes the user's role.

  • Email format validated client-side before submission
  • JWT stored in memory; role decoded to route instantly
  • Invalid credentials surface a clear error message
  • Protected routes redirect unauthenticated users

Drill into any enrolled course to see individual lessons, estimated read times, and completion state. Every toggle fires a live PATCH request to the backend.

  • Course completion % recalculates in real-time
